Диаграмма использования вариантов
```
@startuml "Практическая работа 0"
left to right direction 
skinparam packageStyle rect

actor "Ответственный за заполнение табеля" as tabel
actor "Руководитель подразделения" as boss
actor "Документовед" as doc

rectangle "Табель"{
    (Создать табель за весь месяц) -- tabel
    (Создать табель за первую половина месяца) -- tabel
    (Создать корректирующий табель) -- tabel
    (Отправить табель на согласование) -- tabel
    (Удалить табель) -- tabel
    (Вернуть табель на редактирование) -- tabel

    (Добавить ответственного за заполнение) -- boss
    (Удалить ответственного за заполнение) -- boss
}

rectangle СОУД {
    boss -- (Поставить визу на табель)
    doc -- (Распечатать табель)
    doc -- (Зарегистрировать табель)
}

tabel<|-- boss

@enduml
```
Диаграмма классов
```
@startuml


tabel-->Табель:удаление
tabel-->Табель:создание
tabel-->Табель:редактирование
tabel-->Табель:отправка
boss-->Табель:добавление
boss-->Табель:удаление
boss-->СОУД:виза
Документовод-->СОУД:печать
Документовод-->СОУД:регистрация

class "Ответственный за заполнение табеля" as tabel{
  ФИО
  Номер телефона
  Адрес проживания
  Удаление табеля()
  Создание табеля()
  Редактирование табеля()
  Отправка табеля на согласование()

}

class Табель {
  Номер табеля
  Название
  Ответственный за табель
  Руководитель подразделения
}

class "Руководитель подразделения" as boss {
  ФИО
  Номер телефона
  Добавление ответственного()
  Удаление ответственного()
  Поставить визу на табель()
}

class СОУД{
  Наименование табеля
  Дата табеля
  Номер табеля
}


class Документовод {
  ФИО
  Номер телефона
  Печать табеля()
  Регистрация табеля()
}

@enduml
```
Диаграмма последовательности
```
@startuml
actor "Ответственный за заполнение табеля" as tabel
actor "Руководитель подразделения" as boss
actor "Документовед" as doc
group Создание табеля
tabel-> Табель: Создание
activate Табель
tabel-> Табель: Удаление
tabel-> Табель: Редактирование
Табель -> tabel: Результат табеля
deactivate Табель
activate tabel
tabel-> Табель: Отправка на проверку
deactivate tabel
end

tabel-> boss: Запрос на эл.подпись табеля
activate boss
boss-> doc: Передача табеля
boss -> doc: Эл.подпись табеля
deactivate boss

activate doc
doc->Табель:Печать табеля
doc->Табель: Регистрация табеля
deactivate doc



@enduml
```
